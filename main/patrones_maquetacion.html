Patrones de diseño web

Mobile First/Only consiste en apilar las columnas verticalmente conforma valla teniendo menos espacio la pagina web, eso se ve claramante si vas de desktop a mobile.
Una buena practica de este proceso es mandar a llamar cual archivo css voy a tomar desde el html, sin meternos con el archivo CSS, tendra como propiedad la media query donde sera el punto de interrupcion.
Un ejemplo de este patron 

            link href="nombreArchivo.css" rel="stylesheet" 
 
            media="screen and (min-width: 370px)"

Esta es otro patron de diseño, realiza un desplazamiento del contenido, a diferencia de Mostly Fluid este tiene la idea de ordenar diferente las areas de tu contenido sin colocar una debajo de la otra. Este tiene mas puntos de ruptura.
Un ejemplo de este patronGitHub

Para este patron, el contenido se apilan verticalmente hasta donde tu decidas que para esa fila existen los sufientes contenedores por consiguiente crearias otra fila y se trabaja igual, apilando verticalmente.
Un ejemplo de este patron smashing magazine

Este patron de diseño web aplica para paginas de una sola columan, como son paginas informativas o articulos con mucho texto, los cambios mas usuales con este son menores, como tamaño de fuente, tamaño de imagene, cambios que no descontrolen la armonia de paginas

Este muestra u oculta contenido, dependiendo el tamaño de pantalla, este cambio es mas visible en pantallas pequeñas ya que se tiene como idea principal es mostrar informacion importante para el usuario, en este patron puede ocultar por ej. imagenes o submenus, en vez de ver todo el menu principal tener una ambuerguesa para ocultar este campo.
